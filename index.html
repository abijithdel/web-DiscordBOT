Command's

/ban (username) Bans a user from the Server

/unmute (username) Unmutes a user in the server

/warnings (username) Displays all the past warnings of a user

/status Change the status or presence of the bot, only premium users can change their bot's presence

/invite Generates the invite link for the bot and posts it in the server

/say (message) Broadcasts a message in the channel

/removerole (member) (role) Removes a role from a user

/purge (number) Deletes a number of messages in a channel

/timeout (user) (duration) Gives a user a time-out

/avatar Get a user's avatar

/undeafen (user) Undeafens a user

/slowmode (time) Changes a channel's slowmode

/vmute (user) Voice mutes a user

/unban (user) Unbans a user

/mute (user) Mutes a user in the server

/warn (user) (reason) Warns a user

/clearwarnings (user) Clears all the warnings of a user

/report (user) (reason) Allows users to report an another user, the reports will be stored in a channel named 'reports

/kick (user) Kicks a user from the server

/addrole (member) (role) Adds a role to a user

/softban (user) Soft bans a user from the server, removes all their messages from chat and kicks them

/ping Gets the ping of the server and the bot

/removetimeout (user) Removes a time-out from a user

/deafen (user) Deafens a user

/move (user) (channel) Moves a user to a voice channel

/vkick (user) Voice kicks a user

/vunmute (user) Voice unmutes a user
